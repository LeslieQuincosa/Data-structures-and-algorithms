Readme

El proyecto fue elaborado en C++
Compilador: Dev-C++

El programa no requiere datos de consola.
Solamente se debe correr el programa y en pantalla insertamos el valor de las coordenadas:
x e y.

Nota: dependiendo de la posición inicial del caballo el tiempo de ejecución es más lento o rápido.

Ejemplo: x=4 y=4 es rápido. x=1 y=2 es lento
